# bottega.config — per-project values

> Single source of truth for the placeholders used across the templates. Fill this
> in once (by hand, or let `/bootstrap` populate it by interviewing you), then
> substitute the values into the copied templates. Keep this file in your project
> root; it documents how your Bottega is configured.

| Placeholder | Meaning | Example |
|---|---|---|
| `{{PROJECT_NAME}}` | The project's name | `Acme Notes` |
| `{{PROJECT_ONE_LINER}}` | One sentence: what it is, for whom | `A markdown notes app for small teams` |
| `{{PROJECT_ROOT}}` | Absolute path to the repo on disk | `C:\Users\me\Desktop\Projects\acme-notes` |
| `{{REPO_URL}}` | Git remote URL | `https://github.com/me/acme-notes.git` |
| `{{REPO_VISIBILITY}}` | `private` or `public` | `private` |
| `{{SECRETS_GLOB}}` | Glob(s) for files that must NEVER be committed | `**/.env.local, secrets/**` |
| `{{IP_TO_PROTECT}}` | The moat — proprietary thing that never leaves the private repo (or `none`) | `none` |
| `{{DEPARTMENTS}}` | The persistent departments you enabled | `execution, validation` |
| `{{COMMIT_COAUTHOR}}` | Trailer appended to commits (or empty) | `Co-Authored-By: Claude <noreply@anthropic.com>` |
| `{{MANAGER_MODEL}}` | Model for the manager session | `Opus` |
| `{{COMMITTENTE_LANG}}` | Language the manager speaks to you | `English` |
| `{{STACK}}` | Tech stack (drives the static-check commands) | `Next.js (web) + FastAPI (api)` |
| `{{STATIC_CHECKS}}` | The commands that gate "ready to test" | `npx tsc --noEmit; npm test; pytest` |
